# 问题 1

是不是Ax=0叫齐次方程组，Ax=b就是非齐次方程组呀？怎么理解齐次？

是的，你这个理解在**线性代数里是完全正确的**。

一般我们讨论的是线性方程组
Ax = b

当 **b = 0 向量** 时，就叫
Ax = 0 —— **齐次线性方程组**

当 **b ≠ 0 向量** 时，就叫
Ax = b —— **非齐次线性方程组**

关键就在右边是不是“零”。

---

那“齐次”到底是什么意思？可以从三个层面来理解。

第一，从**形式**上理解。
齐次的意思就是：方程里没有“常数项”。
所有项都跟未知量 x 有关，右边是 0。
所以 Ax = 0 没有任何“外加的量”，而 Ax = b 相当于被外力 b 推了一下。

---

第二，从**解的性质**来理解（这是最重要的）。

齐次方程组 Ax = 0 **永远至少有一个解**：
x = 0（零向量）

而且如果 x₁、x₂ 是解，那么
x₁ + x₂ 还是解，
任意数 λ·x₁ 还是解。

也就是说：
**齐次方程组的解集是一个向量空间**。

非齐次 Ax = b 就不一样了：
• 可能没解
• 即使有解，也不包含 0
• 解集不是向量空间，只是“平移过的东西”

---

第三，从**几何直观**来理解。

在二维或三维空间里：
Ax = 0 描述的通常是
“过原点的直线 / 平面 / 子空间”

Ax = b 描述的是
“不过原点的直线 / 平面”

所以“齐次”本质上就是：
**一切都围绕原点，没有偏移量**。

---

可以用一句话记住：

> 齐次 = 右边为 0 = 没有常数项 = 解集过原点

如果你愿意，我也可以用一个具体的 2×2 或 3×3 的例子，帮你把“齐次 / 非齐次”的差别画成几何图像来理解。

# 问题2

For
为


$$
A = \begin{bmatrix} 1 & 2 & -1 \\ 0 & 1 & 3 \end{bmatrix},
$$

compute a basis for the null space of $A$.

计算 $A$ 的零空间的基。

要求矩阵 $A = \begin{bmatrix} 1 & 2 & -1 \\ 0 & 1 & 3 \end{bmatrix}$的零空间（null space）的一个基。

设未知向量
$x = (x_1, x_2, x_3)^T$，
则 Ax = 0 等价于线性方程组：

- $x_2 + 2x_2 − x_3 = 0$
- $x_2 + 3x_2 = 0$

先由第二个方程解出$x_2 = −3x_3$

代入第一个方程：
$$
x_3 + 2(−3x_3) − x_3 = 0\\
x_1 − 6x_3 − x_3 = 0\\
x_1 = 7x_3
$$

令自由变量 $x_3 = t$，则
$$
x_1 = 7t\\
x_2 = −3t\\
x_3 = t\\
$$
因此零空间中的一般向量为$t(7, −3, 1)^T$。

所以，矩阵 A 的零空间的一组基是${ (7, −3, 1)^T }$。
